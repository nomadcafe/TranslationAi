# Translation AI — 简体中文简介

Translation AI 是基于 **Next.js** 构建的 AI 多语言翻译与内容处理平台：支持**文本**、**图片 OCR**、**PDF**、**语音**与**视频**等能力，可在后端接入多种大模型 API；提供 **NextAuth**（邮箱、GitHub、Google）与 **Stripe** 订阅及按功能划分的用量配额。产品界面与文案支持**英语、中文、日语、西班牙语**。正式环境站点：<https://translation.ai>。开源协议：**MIT**（见仓库根目录 `LICENSE`）。

**完整功能清单、技术栈、环境变量与常用命令**见仓库根目录的 [README.md](README.md)（英文）。
