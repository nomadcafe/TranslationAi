# Translation Ai — 日本語での概要

Translation Ai は **Next.js** ベースの AI 多言語翻訳・コンテンツ処理プラットフォームです。**テキスト**、**画像 OCR**、**PDF**、**音声・動画**に対応し、複数の生成 AI API をサーバー側から利用できます。**NextAuth**（メール、GitHub、Google）と **Stripe** による課金・クォータ管理に対応しています。UI 言語は**英語・中国語・日本語・スペイン語**です。本番サイト：<https://translation.ai>。ライセンスは **MIT**（リポジトリ直下の `LICENSE`）。

**機能一覧・技術スタック・環境変数・スクリプト**の詳細は、リポジトリルートの [README.md](README.md)（英語）を参照してください。
